HTML is a hierarchical language built to use tags as separators and create increasingly specific section to add content to. It wil begin with the over all html tag then more specific ones like head, body, and footer to denote the top middle and bottom of the page. Then you have more specific HTML tags like the paragraph tag to denote a large amount of text content or headings 1 through 6 which can be used to mark sections and subsections.
