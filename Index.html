Basic Introduction

HTML, or the HyperText Markup Language, is the standardized language for creating and structuring a webpage on the Internet. It is used to add and define various pieces of content that a person can put on a webpage, designed to be the foundation that other languages would work with or build off of. HTML documents have a certain structure to them that gradually gets more specific as you use more tags, which allows the user to define the content type as well as the general location, giving the webpage some initial organization and structure.

Many of the tags in HTML consist of an opener and a closer that define what elements are inside of it and what is not. In addition to this some contain attributes that one can use to add additional information for what is contained inside of it. It can be used to add more structure, define their purpose, add styling, add destinations and sources, and more.
